# Str To Img

URL クエリパラメーターで与えた文字列を画像に変換します.

サンプル画像
<img src="./sample.png"/>

## 使い方

このリポジトリをクローンします

```
git clone https://github.com/kajikentaro/str-to-img
cd str-to-img
```

以下コマンドで実行します

```
go run .
```

ブラウザから  
http://localhost:8080?text=テキスト  
のようにアクセスします

## 使い方(docker compose)

以下コマンドを実行します

```
docker compose up
```
